# DGHNet: Discrepancy Learning Guided Hierarchical Fusion Network for Multi-modal Recommendation

## Introduction

This is the Pytorch implementation for our KBS paper:

> Dang Y, Pan Z, Zhang X, et al. Discrepancy Learning Guided Hierarchical Fusion Network for Multi-modal Recommendation[J]. Knowledge-Based Systems, 2025, 317: 113496.

## Environment Requirement

- Python 3.9
- Pytorch 2.5.1+cu124

## Dataset

The original dataset is the publicly available [Baby/Sports/Clothing](http://jmcauley.ucsd.edu/data/amazon/links.html) datasets from Amazon.
